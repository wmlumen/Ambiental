Arquitectura del portal

Stack tecnológico

Estructura de rutas

/
├── cursos
│   └── [course]                  (primer-curso, segundo-curso, tercer-curso)
│       └── [subject]             (asignatura)
│           └── [unit]            (unidad didáctica)
│               ├── leccion-01-fundamentos
│               └── leccion-02-aplicacion
├── mapa-web                      (mapa completo del sitio)
└── arquitectura                  (esta página)

Despliegue

El portal se compila como sitio estático con output: 'export' y se publica automáticamente en GitHub Pages mediante GitHub Actions. Cada push a la rama maindispara el build y el deploy a la rama gh-pages.
